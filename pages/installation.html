Installation de l'environement

WampServer

  • WampServer est une plate-forme de développement Web sous Windows pour des applications Web dynamiques à l’aide du serveur Apache2, du langage de scripts PHP et d’une base de données MySQL. Il possède également PHPMyAdmin pour gérer plus facilement vos bases de données.
    Télécharger WampServer
  • Pour que WampServer fonctionne sous Windows il faut installer les Paquetages VC++
    Télécharger Visual C++ Redistributable Packages for Visual Studio 2015-2022 x86
    Télécharger Visual C++ Redistributable Packages for Visual Studio 2015-2022 x64
  • Configurer les variables d'environement pour WampServer
    Dans Windows Faire "Recherche" > "Modifier les variables d'environnement système" > Cliquer sur "Variables d'environnement" >
    Dans "la liste des Variables système" > Double cliquer sur la ligne "Path" puis verifier qu'il y a bien "C:\wamp64\bin\php\php8.2.0". (sinon l'ajouter)


Télecharger et installer Composer

  • Composer est un gestionnaire de dépendances pour PHP. Il permet d’ajouter et de gérer des dépendances. C’est-à-dire des librairies externes, déjà codées par d’autre développeurs et que l'on peux intégrer à ses projets. > Télécharger Composer


Installer Scoop & Symfony CLI

  • Ouvrir le Terminal Windows PowerShell. Lancer cette première ligne de commande : 
    Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
  • Puis lancer cette seconde ligne de commande : 
    Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression
  • Ensuite installer Symfony CLI : 
    scoop install symfony-cli


Verifications de l'installation avec GitBash:

  • Tester Composer 
    composer
  • Tester PHP 
    php -v
  • Tester Symfony 
    symfony check:requirements
