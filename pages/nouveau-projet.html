Créer un nouveau projet

Lancer GitBash dans le répertoire ou sera crée le nouveau projet (ex: C:\wamp64\www)

  • Remplacer "my_project_directory" par le nom de votre nouveau dossier/projet, puis lancer la commande : 
    symfony new my_project_directory --version="7.0.*" --webapp
  • Se rendre dans le dossier nouvellement crée : 
    cd my_project_directory/
  • Indiquer (par prudence) quelle version de PHP sera utilisée : 
    echo 8.2 > .php-version
  • Démarrer le serveur local PHP de Symfony : 
    symfony server:start
  • Se rendre à l'adresse web local du projet nouvellement crée (en principe : http://127.0.0.1:8000) 
    http://127.0.0.1:8000
